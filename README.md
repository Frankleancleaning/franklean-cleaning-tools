# Franklean Cleaning Tools

Simple static web tools for Franklean Cleaning in Brisbane.

## Included

- Quote calculator for bond cleaning, carpet steam cleaning, and pest control
- Confirmation email generator
- Short SMS generator
- Copy buttons for customer messages, emails, and SMS replies

## How To Run

Open `index.html` in a web browser.

No installation, server, build step, or React setup is required.

## Business Notes

- Prices are starting prices only.
- Final price must be confirmed after inspection or discussion before cleaning starts.
- Final price will be discussed and agreed on the day before cleaning starts for booking confirmations.
- Payment must be completed after final price agreement and before cleaning starts.
- No deposit is required unless specified.
- Extra charges may apply depending on property condition.

## GitHub Pages

To publish with GitHub Pages:

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, `app.js`, and this `README.md`.
3. In GitHub, open `Settings` then `Pages`.
4. Set the source to the main branch and root folder.
5. Save, then open the GitHub Pages URL after it finishes publishing.

## Customising Prices

Edit the `prices` object near the top of `app.js`.

The calculator intentionally keeps pricing simple so starting rates can be adjusted quickly.
